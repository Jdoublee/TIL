# 뷰와 컨텐츠
 
## 문자 기반 컨텐츠
- 뷰 : **UILabel**
- 컨텐츠 : **String**

1. 글자가 필요로 하는 크기
- Size to Fit Content
2. 문자열과 레이블 크기 다른 경우
- Line Break
- Autoshrink
- Lines

### 폰트
- 폰트 클래스 : **UIFont**
- 시스템 폰트 추가시 _plist_ 수정

## 이미지 기반 컨텐츠
- 뷰 : **UIImageView**
- 컨텐츠 : **UIImage**


### 이미지 사용 사례
  - 접근 방법 달라짐
1. 화면 UI (일부) 이미지
- 앱과 함께 배포 (bundle)
- 파일/애셋
2. 사용자 컨텐츠 이미지


### 이미지 리소스
1. 파일
- 기기에 맞는 이미지 자동 적용
  - 비 레티나 모델 : 이미지 이름
  - 레티나 모델 : 이미지 이름@2x
  - 레티나 모델(+) : 이미지 이름@3x

2. 애셋

: 프로젝트 리소스 관리 

- 앱 아이콘
- 이미지 셋
- 런치 이미지
- 데이터
